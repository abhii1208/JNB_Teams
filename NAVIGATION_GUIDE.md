# Quick Navigation Guide - After Redesign

## Where Everything Is Now

### 🏢 **Workspace Selection**
**Location**: Top-right corner → Profile Avatar → Workspace List
- Click your profile avatar
- See all your workspaces with role badges
- Click any workspace to switch
- Use "Create Workspace" button to add new

### 📊 **Dashboard**
**Location**: Sidebar → Dashboard
- Shows stats for **current workspace**
- Displays: Active Projects, Tasks Due Today, Pending Approvals, Completion Rate
- Recent tasks and project progress

### 📁 **Projects**
**Location**: Sidebar → Projects
- Lists all projects in **current workspace**
- Click project to see details
- Project stats: Open Tasks, Pending Approval, Completed, Progress
- Tabs: Tasks, Members, Settings

### ✅ **Tasks**
**Location**: Projects → Select Project → Tasks Tab
- **3 View Modes**:
  - **List View**: Grouped by status (Pending → Open → Completed)
  - **Board View**: Kanban columns (Planned → In-process → Completed → On-hold → Dropped)
  - **Calendar View**: Monthly calendar with tasks on due dates
- Switch views using toggle buttons
- Click any task to view details

### 👥 **Team**
**Location**: Sidebar → Team
- Shows team members in **current workspace**
- View roles and manage members
- Add new members (Owner/Admin only)

### ⚙️ **Settings**
**Location**: Top-right → Profile Avatar → Settings
- Profile settings
- Notification preferences
- Appearance and language
- Security options

### 🔔 **Notifications**
**Location**: Top-right → Bell Icon
- Shows notification count badge
- Click to see recent notifications
- Quick access to updates

### 🚪 **Logout**
**Location**: Top-right → Profile Avatar → Logout
- Click profile avatar
- Click "Logout" at bottom of menu

## Common Tasks

### Switching Workspaces
1. Click profile avatar (top-right)
2. Click desired workspace
3. Dashboard and projects update automatically

### Viewing Tasks in Different Ways
1. Go to Projects → Select Project
2. Click Tasks tab
3. Choose view: List | Board | Calendar
4. Tasks automatically organize by selected view

### Creating New Task
1. Projects → Select Project
2. Click "New Task" button
3. Fill in task details
4. Task appears in all views

### Checking Notifications
1. Click bell icon (top-right)
2. View recent notifications
3. Click notification to navigate

### Managing Project Settings
1. Projects → Select Project
2. Click "Settings" tab or button
3. Configure permissions and workflows

## Tips & Shortcuts

💡 **Workspace Context**: Current workspace shown in center of top bar - always visible
💡 **Quick Navigate**: Use sidebar for main sections (Dashboard, Projects, Team)
💡 **Task Views**: Use List for status, Board for stage, Calendar for scheduling
💡 **Profile Menu**: One-stop for workspaces, settings, and logout
💡 **Stats Overview**: Dashboard shows aggregated data for current workspace only

## Mobile Considerations

On smaller screens:
- Sidebar may collapse to hamburger menu
- App bar remains visible and fixed
- Calendar view adapts to vertical layout
- Board view becomes horizontally scrollable
